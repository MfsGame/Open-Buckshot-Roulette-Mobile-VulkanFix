# Open Buckshot Roulette-Mobile Vulkan fix
这是一个把 Buckshot Roulette (恶魔轮盘) 学习项目  
如果你想修改或使用 Open Buckshot Roulette 请确保 Buckshot Roulette 你在 itch.io 或 Steam 账户上可用
